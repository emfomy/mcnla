### Makefile

| Command            | Detail                   |
|--------------------|--------------------------|
| `make all`         | build all binaries       |
| `make doc`         | build documentation      |
| `make run`         | run demo code            |
| `make check`       | build and run test codes |
| `make build-test`  | build test codes         |
| `make help`        | display make-rules       |
